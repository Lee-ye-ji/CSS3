영역 관련 속성

1) content : 내용 표시되는 영역
2) border : 요소의 테두리 영역
3) padding : content와 border 사이의 영역 4) margin : 타 요소와의 간격을 지정하는 영역
현재 div의 테두리를 포함한 가로 길이
= width + 양쪽 padding + 양쪽 border = 300 + (20 * 2) + (20 * 2)
= 380
현재 div의 테두리를 포함한 세로 길이
= height + 위아래 padding + 위아래 border
= 200 + (20 * 2) + (20 * 2)
= 280px
